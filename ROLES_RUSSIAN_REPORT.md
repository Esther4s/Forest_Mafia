## 🔧 **ТЕХНИЧЕСКИЕ ДЕТАЛИ**

### **Созданный модуль `role_translator.py`:**
```python
def get_role_name_russian(role: Role) -> str:
    """Возвращает русское название роли"""
    role_names = {
        Role.WOLF: "Волк",
        Role.FOX: "Лиса", 
        Role.HARE: "Заяц",
        Role.MOLE: "Крот",
        Role.BEAVER: "Бобёр"
    }
    return role_names.get(role, "Неизвестно")
```

### **Переводы ролей:**
- `Role.WOLF` → "Волк"
- `Role.FOX` → "Лиса"
- `Role.HARE` → "Заяц"
- `Role.MOLE` → "Крот"
- `Role.BEAVER` → "Бобёр"
